### Ключевое открытие: ЭТО НЕ FRONT-RUNNING

Спот (PALMAI) и фьючерсы (STO/PIPPIN/SIREN) — **разные монеты**. Это значит:
- Wash-trade на PALMAI не создаёт движение для front-run на STO
- Фьючерсный скальпинг — **самостоятельная стратегия** (market-making/scalping на microstructure)
- Спот wash-trade — **отдельная цель** (возможно: объём для получения rewards, airdrop farming, или просто убыточная операция)

Учитывая **наши знания о SyncTrade** (зеркальные позиции на двух аккаунтах), наиболее вероятная схема:

1. **Этот аккаунт** = хедж-сторона (убыток −51 USDT по фьючерсам)
2. **Второй аккаунт** (незнакомый нам) = мастер-сторона (прибыль ~+51 USDT)
3. **Фьючерсный скальпинг** — не самостоятельный, а часть SyncTrade-пары
4. **2,777 USDT переводов** — вероятно, профит с мастер-аккаунта, перекинутый через промежуточные каналы

**Доказательства за SyncTrade:**
- Win rate 80% но ИТОГО убыток — типично для хедж-стороны synctrade если counter-party берёт крупные противоположные позиции

- Нулевые комиссии (54/100 позиций) — MEXC zero-fee maker
- Позиции по 1-30 секунд — типичный SyncTrade timing

- Нужен начальный депозит ~50-100 USDT на обоих аккаунтах

### Оценка сложности: **средняя**
- Спот-бот: простой (buy/sell mkt чередование) — 1 день разработки
- Фьючерсный бот: сложнее (scalping + order management) — 2-3 дня
