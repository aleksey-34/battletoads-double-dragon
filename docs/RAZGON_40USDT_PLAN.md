## 3. Стратегия: «Тройной разгон»

Комбинируем три подхода параллельно для максимизации оборота и диверсификации риска.

### 3.1 Ядро — Momentum Scalping (60% капитала = ~24 USDT)

**Логика:** Ловим краткосрочные импульсы (1-15 мин) на 1m/5m таймфреймах.

| Параметр | Значение |
|---|---|
| Плечо | 20-30x |
| Размер позиции | 4-8 USDT маржи (80-240 USDT нотионал) |
| Макс одновр. позиций | 2-3 |
| Take Profit | 0.3-0.8% от нотионала (trailed) |
| **Stop Loss** | **0.15-0.3%** (ЖЕЛЕЗНЫЙ, рыночный) |
| RR | 1:2 минимум |
| Сделок в день | 20-50 |
| Ожидаемый WR | 55-65% |

**Сигналы входа:**
1. **Volume spike** — объём текущей 1m-свечи > 3x средний за 20 баров → вход по направлению свечи
2. **Donchian micro-breakout** — пробой 15-барного (15 мин) максимума/минимума на 1m
3. **RSI divergence** — RSI(14) на 5m расходится с ценой → контр-тренд только если совпадает с зоной поддержки/сопротивления

**Управление позицией:**
```
Вход: Market order по сигналу
SL: −0.2% от нотионала (СРАЗУ при открытии, limit/market)
TP: Trailing — anchor двигается за ценой, стоп на −0.3% от anchor
Макс. время в позиции: 15 мин (принудительное закрытие)
```

**Почему это лучше чем MEXC-бот:**
- **Жёсткий SL** ограничивает макс. убыток до ~0.5-1.5 USDT (а не −78 как у client1)
- **Таймаут позиции** — не зависаем в убыточной позиции часами
- **Donchian micro-breakout** — уже реализован в нашем engine.ts, нужно адаптировать под 1m

### 3.2 Ракета — News/Listing Sniper (25% капитала = ~10 USDT)

**Логика:** Быстрый вход при листингах, анонсах, Binance Alpha сигналах.

| Параметр | Значение |
|---|---|
| Плечо | 10-20x |
| Размер | 5-10 USDT маржи |
| Макс позиций | 1 |
| TP | 3-10% (быстрый выход) |
| SL | 2% |

**Триггеры:**
- Новый листинг на Bitget → первые 5 мин максимальный рост
- Binance анонс → альт реагирует на других биржах с задержкой 30-120 сек
- Twitter/X крипто-инфлюенсеры → мем-коины реагируют за 1-5 мин

**Реализация:**
- Мониторинг Bitget announcements API (или парсинг)
- Мониторинг Binance websocket для новых пар
- Готовые ордера для быстрого входа

### 3.3 Пассив — Funding Rate Farming (15% капитала = ~6 USDT)

**Логика:** Удержание позиции в направлении получения funding fee. Bitget funding каждые 8ч.

| Параметр | Значение |
|---|---|
| Плечо | 5-10x |
| Размер | 6 USDT маржи |
| TP | Нет (держим пока funding положительный) |
| SL | 3% |

**Монеты:** Те, где funding rate > 0.05% (>0.15%/день = ~55%/год).

На MEXC client1 заработал +93 USDT за 3 месяца чисто на фандинге. С $6 на 10x = $60 нотионал, при 0.1% funding = $0.06 каждые 8 часов = ~$5.4/мес. Мало, но **бесплатные деньги** пока основные стратегии работают.

## 4. Управление рисками — КРИТИЧНО

### Правила, которые нельзя нарушать

| # | Правило | Обоснование |
|---|---------|-------------|
| 1 | **Макс. убыток на сделку ≤ 5% депозита** ($2) | При WR 55% и RR 1:2 — математическое преимущество |
| 2 | **Макс. дневной убыток ≤ 20%** ($8) | Стоп на день — предотвращает тильт |
| 3 | **Изолированная маржа ВСЕГДА** | Не рискуем всем балансом на одной позиции |
| 4 | **Стоп-лосс ДО открытия позиции** | Размещаем SL как часть ордера (OCO/bracket) |
| 5 | **Не держать позицию >15 мин (скальпинг)** | Таймаут предотвращает "застревание" |
| 6 | **Не увеличивать убыточную позицию (no averaging down)** | Это уничтожило MEXC-бота |
| 7 | **Рескейл позиции каждые +25% к депо** | $40→$50: size +25%, $50→$62: size +25%... |

### Сценарии роста

```
День 1-3:  $40 → $55  (+37%)  — 15 сделок/день × $1 avg profit
День 4-7:  $55 → $80  (+45%)  — увеличенные позиции
День 7-10: $80 → $130 (+62%)  — compound effect
День 10-14: $130 → $250+ (+92%) — если WR держится

Пессимистичный: $40 → $25 (drawdown 37%, стоп и пересмотр)
Реалистичный:   $40 → $80-120 за 2 недели
Оптимистичный:  $40 → $300+ если поймаем 2-3 listing pump'а
```

## 6. Мировые тенденции и хитрости (апрель 2026)

### Что сейчас работает на рынке

| Тренд | Как использовать |
|---|---|
| **AI-агенты в крипто** | Мем-коины на тему AI (GOAT, VIRTUAL, AI16Z) — волатильность 20-50%/день |
| **Bitget Copy Trading** | Если разгоним — можно получить фолловеров и комиссию с их прибыли |
| **Funding rate дисбалансы** | После крупных движений BTC funding rate на альтах взлетает до 0.3-1% → short farming |
| **Binance Alpha → Bitget listing** | Монеты из Binance Alpha через 1-3 дня листятся на Bitget — можно покупать заранее |
| **Weekend volatility** | Выходные — меньше ликвидности, больше ложных пробоев → scalping более эффективен |
| **Liquidation cascades** | При BTC ±3% — каскадные ликвидации на альтах, пики волатильности для скальпинга |

### Хитрые тактики

1. **Dual-account mirror (SyncTrade):** Открываем long на одном аккаунте, short на другом. Один всегда в профите. Выводим профит, закрываем оба. Работает при низких комиссиях (Bitget maker 0.02%). Но нужен **второй аккаунт** и ~$40 второй.

2. **Grid + Leverage:** Сетка ордеров с плечом 10x на боковом рынке. При $40 маржи = $400 нотионал. Сетка из 10 уровней по $40 каждый. Доход 0.1-0.5% на каждом цикле grid'а.

3. **Momentum riding:** Определяем монету, которая за последний час выросла >5%. Входим long на откате (pullback к VWAP). При $40 на 20x = $800 нотионал. TP 1% = $8 (20% к депо). SL 0.5% = $4 (10% к депо).
