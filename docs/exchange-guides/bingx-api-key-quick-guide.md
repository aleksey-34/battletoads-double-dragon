# BingX API Key Quick Guide

## Где создать ключ
- Откройте `BingX -> API Management`.
- Создайте API-ключ для деривативов.

## Какие права включить
- `Read`.
- `Trade`.
- Не включайте права вывода (`Withdraw`).

## Что скопировать
- `API Key`.
- `Secret Key`.

## Whitelist
- Добавьте IP вашего сервера в whitelist.
- Проверяйте, чтобы IP был без лишних пробелов и с корректным форматом.

## Что выбрать в ЛК
- В форме ключа выберите `BingX Futures`.
- Вставьте `API Key` и `Secret`.
- `Passphrase` обычно не требуется.

## Быстрый чек
- Права только `Read + Trade`.
- `Withdraw` выключен.
- IP whitelist включен.
