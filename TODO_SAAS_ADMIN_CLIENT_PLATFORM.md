# TODO SaaS-платформа: Админ + Клиент

## Цель
Собираем три режима в одном продукте:
- Админ-режим: максимум функций (клиенты, оплата, тарифы, стратегии, полный бектест, нагрузка VPS/API).
- Клиентский режим стратегий: упрощенный интерфейс со стратегиями, оплатой, лимитами и безопасной автоматикой.
- Режим `Алгофонд`: клиент видит только мониторинг единой торговой системы, описание/эквити и подает запросы на старт/стоп.
