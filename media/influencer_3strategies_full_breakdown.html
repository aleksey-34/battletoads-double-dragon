Полный расклад по офферам для инфлов

Ниже не "витринные лозунги", а расчетные метрики по бэктест-снапшотам (90d) из runtime-хранилища VPS + отдельная карточка новой TS на объем. Комиссия для модели: 0.12% round-trip, депозит: $100,000, средняя загрузка капитала: 25%.

Это именно "TS на объем": максимальная частота входов и высокий оборот. Главный лимитер маржи — комиссия и микрослип.

3) Важные выводы для инфл-оффера

  • Conservative лучше продавать как low-turnover/стабильный профиль.
  • Balanced сейчас самый сильный по связке PF + частота + средний профит на сделку.
  • Aggressive/Volume даёт высокий оборот, но имеет самый жесткий fee-burden.
  • Для объёмной TS нужно заранее проговаривать, что комиссия за период может "съедать" значимую часть валовой прибыли.

Прод фикс materialization уже в проде: retry-materialize больше не падает на pair-conflict, backend path стабилен.
